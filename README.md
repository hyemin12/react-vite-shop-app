# 타입스크립트기반의 쇼핑몰 사이트

[사이트](https://react-shop-app-4692d.firebaseapp.com/)로 이동하기

■ 패키지 모듈 설치  
`$npm i`

■ 개발 서버 열기  
`$npm run dev`

- react-hook-form
- firebase (auth/hosting)
